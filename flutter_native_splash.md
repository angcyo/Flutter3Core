# flutter_native_splash

`flutter_native_splash: ^2.4.7`

https://pub.dev/packages/flutter_native_splash

## 添加依赖

```yaml
dev_dependencies:
  flutter_native_splash: ^2.4.7
```

## 使用指定配置文件生成启动图像`splash screen`

```shell
dart run flutter_native_splash:create --path=Flutter3Core/flutter_native_splash.yaml
```

```shell
dart run flutter_native_splash:create --path=../../Flutter3Core/flutter_native_splash.yaml
```
