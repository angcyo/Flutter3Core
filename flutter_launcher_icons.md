# flutter_launcher_icons

`flutter_launcher_icons: ^0.14.3`

https://pub.dev/packages/flutter_launcher_icons

## 添加依赖

```yaml
dev_dependencies:
  flutter_launcher_icons: ^0.14.3
```

## 使用指定配置文件生成图标

```shell
dart run flutter_launcher_icons -f Flutter3Core/flutter_launcher_icons.yaml
```

```shell
dart run flutter_launcher_icons -f ../../Flutter3Core/flutter_launcher_icons.yaml
```
