关于火焰
Flame 是一个模块化的 Flutter 游戏引擎，它为游戏开发提供了一整套简洁高效的解决方案。它充分利用了 Flutter 强大的基础架构，同时简化了构建项目所需的代码。

它提供了一个简单而有效的游戏循环实现方案，以及游戏中可能需要的必要功能。例如：输入、图像、精灵、精灵图、动画、碰撞检测，以及我们称之为火焰组件系统（简称 FCS）的组件系统。

我们还提供独立的软件包，用于扩展 Flame 的功能，这些软件包可以在以下位置找到： 桥梁套餐部分。

您可以根据需要自由选择所需的部件，因为它们都是独立且模块化的。

引擎及其生态系统正不断受到社区的改进，因此请随时联系我们，提出问题和提交 PR，并提出建议。

如果你想帮助提升引擎的曝光度并壮大社区，请给我们点个赞吧！:)
